My main Windows 98 computer

Specifications

  • Processor: Pentium 4 @ 1.50 GHz, socket 423
  • RAM: 256MB of RDRAM
  • Graphics card: NVidia Geforce FX 5900XT
  • Operating System: Windows 98SE

About this computer

This PC uses one of the very first P4 procssors, which means that it's one of the fastest period-appropriate Windows 98 machines.
It also uses RDRAM (like the N64!), which, while faster than normal SDRAM, was more expensive and definitely not as easy to find nowadays.
